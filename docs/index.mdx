# Startups.org.ai

Welcome to the Startups.org.ai documentation. This monorepo contains a collection of AI-powered tools and frameworks for building startups.

## Overview

Startups.org.ai provides a comprehensive suite of packages for:

- **Startup Building**: Tools for ideation, validation, and launching startups
- **Business Frameworks**: Implementations of popular business methodologies (Lean Canvas, StoryBrand, Jobs-to-be-Done, etc.)
- **AI Functions**: Core AI primitives and functions for building intelligent applications
- **Domain Management**: Tools for builder domains and multi-tenant applications

## Getting Started

```bash
# Clone the repository with submodules
git clone --recurse-submodules https://github.com/dot-org-ai/startups.org.ai.git

# Install dependencies
pnpm install

# Start development
pnpm dev
```
