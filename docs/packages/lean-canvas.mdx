# lean-canvas

Lean Canvas business model implementation.

## Installation

```bash
pnpm add lean-canvas
```

## Overview

Create and iterate on Lean Canvas business models:

- Problem identification
- Solution definition
- Unique Value Proposition
- Customer segments
- Channels and revenue streams
- Cost structure
- Key metrics
- Unfair advantage

## Usage

```typescript
import { LeanCanvas } from 'lean-canvas'

const canvas = new LeanCanvas({
  problem: ['Pain point 1', 'Pain point 2'],
  customerSegments: ['Early adopters'],
  uniqueValueProposition: 'Clear, compelling message'
})
```
