# ideal-customer-profile

ICP (Ideal Customer Profile) definition and analysis.

## Installation

```bash
pnpm add ideal-customer-profile
```

## Overview

Define and refine your ideal customer profile:

- Demographic attributes
- Firmographic data
- Behavioral patterns
- Pain points and needs
- Decision-making process

## Usage

```typescript
import { ICP } from 'ideal-customer-profile'

const profile = new ICP({
  industry: 'SaaS',
  companySize: '10-50 employees',
  role: 'Founder/CEO',
  painPoints: ['Slow validation', 'Limited resources']
})
```
