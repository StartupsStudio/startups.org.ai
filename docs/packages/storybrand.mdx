# storybrand

StoryBrand framework implementation for marketing narratives.

## Installation

```bash
pnpm add storybrand
```

## Overview

Build compelling brand stories using the StoryBrand framework:

- Character (your customer)
- Problem (external, internal, philosophical)
- Guide (your brand)
- Plan
- Call to Action
- Success
- Failure

## Usage

```typescript
import { StoryBrand } from 'storybrand'

const story = new StoryBrand({
  character: 'Startup founders',
  externalProblem: 'No clear path to validation',
  internalProblem: 'Fear of wasting time and money',
  guide: 'AI-powered startup tools'
})
```
