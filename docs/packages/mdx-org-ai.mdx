# mdx.org.ai

MDX-based documentation and content management tools.

## Installation

```bash
pnpm add mdx.org.ai
```

## Overview

`mdx.org.ai` provides tools for working with MDX content:

- MDX parsing and transformation
- Content management utilities
- Documentation generation
- Linked Data support (MDX-LD)

## Sub-packages

- `@mdxdb/*` - MDX database packages
- `@mdxld/*` - MDX Linked Data packages
- `@mdxe/*` - MDX extensions
- `@mdxui/*` - MDX UI components
- `@mdxai/*` - MDX AI integrations
